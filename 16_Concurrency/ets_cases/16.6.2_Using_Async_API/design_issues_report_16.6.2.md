# 16.6.2 Using Async API - 设计问题报告

## 1. 概述

当前覆盖 async 函数定义与链式调用的编译验证。async 函数的运行时调度语义（yield、resume）需运行时验证。

## 2. 设计问题清单

| # | 问题类型 | 问题描述 | 建议方案 | 优先级 |
|---|---------|---------|---------|:------:|
| 1 | 覆盖不足 | 缺少 runtime 用例验证 async 函数实际异步执行 | 使用 ark VM 执行 async 调用链 | 高 |
| 2 | 覆盖不足 | async 函数返回 Promise 类型验证不足 | 增补 Promise 类型推导相关用例 | 中 |

## 3. 改进建议

- runtime 测试：验证 async 链中 await 确实挂起执行流
- 编译期测试：验证 async 函数调用非 async 函数的交叉场景
