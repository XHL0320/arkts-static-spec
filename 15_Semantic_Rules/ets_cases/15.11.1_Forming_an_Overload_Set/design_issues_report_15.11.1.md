## 三、设计观察

### 观察：ArkTS 重载集形成与 Java 一致

ArkTS 的重载集形成规则与 Java 类似，都是基于方法名和参数列表（类型、数量、顺序）来区分重载。

### 观察：仅返回类型不同不构成重载

ArkTS 与 Java、Swift 一致，仅返回类型不同不构成重载。这是为了避免调用时的歧义。
