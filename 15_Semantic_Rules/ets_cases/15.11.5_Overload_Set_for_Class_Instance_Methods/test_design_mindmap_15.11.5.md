# 15.11.5 Overload Set for Class Instance Methods - 测试设计思维导图

## 概述
本节定义类实例方法的重载集形成规则：类中可以定义多个同名实例方法形成重载集，通过实例调用实例方法时根据参数类型、数量、顺序选择最匹配的版本。

## 核心规则

### 1. 类实例方法重载集形成条件
- 类中存在多个同名实例方法
- 实例方法参数类型、数量或顺序不同
- 仅返回类型不同不构成重载

### 2. 实例方法重载解析规则
- 通过实例调用实例方法
- 根据调用参数的数量匹配
- 根据调用参数的类型匹配
- 选择最匹配的实例方法版本

### 3. 实例方法重载拒绝规则
- 参数类型不匹配时编译报错
- 参数数量不匹配时编译报错
- 实例方法不存在时编译报错

## 测试点覆盖

### compile-pass
1. 类中定义实例方法重载
2. 通过实例调用实例方法重载
3. 实例方法重载正确解析

### compile-fail
1. 参数类型不匹配拒绝
2. 参数数量不匹配拒绝
3. 实例方法不存在拒绝

### runtime
1. 实例方法重载运行时正确调用
2. 实例方法重载解析运行时验证
