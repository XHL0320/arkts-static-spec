# 15.11.3 Overload Set for Interface Methods - 测试设计思维导图

## 概述
本节定义接口方法的重载集形成规则：接口中可以定义多个同名方法形成重载集，实现类必须实现所有重载方法版本。

## 核心规则

### 1. 接口方法重载集形成条件
- 接口中存在多个同名方法
- 方法参数类型、数量或顺序不同
- 实现类必须实现所有重载方法版本

### 2. 接口方法重载解析规则
- 根据调用参数的数量匹配
- 根据调用参数的类型匹配
- 选择最匹配的方法版本

### 3. 接口方法重载拒绝规则
- 参数类型不匹配时编译报错
- 参数数量不匹配时编译报错
- 实现类未实现所有重载方法版本时编译报错

## 测试点覆盖

### compile-pass
1. 接口中定义重载方法
2. 实现类实现所有重载方法版本
3. 通过接口调用重载方法

### compile-fail
1. 参数类型不匹配拒绝
2. 实现类未实现所有重载方法版本
3. 类型不兼容拒绝

### runtime
1. 接口方法重载运行时正确调用
2. 接口方法重载解析运行时验证

## 编号规划
- compile-pass: 001 ~ 010
- compile-fail: 011 ~ 020
- runtime: 021 ~ 030
