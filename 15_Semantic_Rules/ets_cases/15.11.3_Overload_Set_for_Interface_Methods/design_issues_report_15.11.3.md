## 三、设计观察

### 观察：ArkTS 接口方法重载与 Java 接口方法重载一致

ArkTS 的接口方法重载机制与 Java 的接口方法重载机制类似，都是在接口中定义多个同名方法形成重载集，实现类必须实现所有重载方法版本。

### 观察：接口方法重载的实现要求

实现类必须实现接口中所有重载方法版本，这与 Java 的要求一致。
