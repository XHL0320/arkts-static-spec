# 15.11.7 Overload Set Warning - 测试设计思维导图

## 概述
本节定义重载集的警告机制：当重载集中存在不可达重载（等价签名）时，编译器发出W2323警告，但不阻塞编译。

## 核心规则

### 1. 重载集警告触发条件
- 重载集中存在等价签名（参数类型、数量、顺序完全相同）
- 某个重载永不被选择（不可达重载）
- 编译器发出W2323警告

### 2. 重载集警告行为
- 警告不阻塞编译
- 警告不阻塞运行
- 警告提示开发者修复等价签名问题

### 3. 合法重载集
- 参数类型不同
- 参数数量不同
- 参数顺序不同
- 无等价签名

## 测试点覆盖

### compile-pass
1. 可区分重载无警告
2. 参数类型不同的重载
3. 参数数量不同的重载

### compile-fail（实际为compile-pass，但测试警告）
1. 等价签名重载产生W2323警告
2. 不可达重载警告

### runtime
1. 重载函数运行时正确调用
2. 警告不影响运行时行为

## 编号规划
- compile-pass: 001 ~ 010
- compile-fail（警告）: 011 ~ 020
- runtime: 021 ~ 030
