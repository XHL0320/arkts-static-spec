## 设计规范建议

### 建议 1: 明确联合类型的子类型规则

**现状**: ArkTS 规范未详细说明联合类型的子类型规则

**建议**: 在规范中详细说明以下规则：
1. 联合类型子类型：`U <: T` 当且仅当 `U` 的每个成员 `<: T`
2. 联合类型的身份：`T <: T | U` 和 `U <: T | U`
3. 联合类型的分配性：`A | B <: T` 当且仅当 `A <: T` 且 `B <: T`

**示例**:
```arkts
// 建议：明确联合类型的子类型规则
type A = Dog | Cat;
type B = Animal;
let a: A = new Dog();
let b: B = a; // ✅ Dog <: Animal, Cat <: Animal, 所以 A <: B
```

2. `instanceof` 收窄: `Dog | Cat` 可以通过 `instanceof` 收窄

