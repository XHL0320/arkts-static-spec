## 三、设计观察

### 观察：ArkTS 构造函数重载与 Java 构造函数重载一致

ArkTS 的构造函数重载机制与 Java 的构造函数重载机制类似，都是在类中定义多个构造函数形成重载集，通过new关键字调用构造函数时根据参数列表选择最匹配的版本。

### 观察：构造函数重载的调用方式

ArkTS 通过new关键字调用构造函数（`new ClassName(args)`），这与 Java 的调用方式一致。
