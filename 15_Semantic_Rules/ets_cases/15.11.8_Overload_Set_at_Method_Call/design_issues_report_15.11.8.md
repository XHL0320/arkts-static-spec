## 三、设计观察

### 观察：ArkTS 调用时重载解析与 Java 一致

ArkTS 在每个调用点根据实参形成重载集并解析，这与 Java 的方法调用解析机制类似。

### 观察：调用时重载解析的编译时机

调用时重载解析在编译时进行，根据静态类型信息选择最匹配的重载版本。
