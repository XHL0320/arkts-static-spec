# 15.7.2 Return Type Inference（返回类型推断）- 测试设计思维导图

## 一、测试范围

验证 ArkTS 对函数返回类型的推断行为，包括：
- 函数返回类型匹配（显式返回类型与实际返回值类型一致）
- 返回值协变（子类方法可返回子类型）
- 返回类型不匹配拒绝（声明的返回类型与实际返回值类型不匹配）
- 缺少 return 语句拒绝（非 void 函数缺少 return 语句）
- 返回类型运行时行为
