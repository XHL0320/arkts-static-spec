# 7.13.2 String Indexing Expression - 测试设计思维导图

## 概述

字符串索引表达式 `s[i]` 是访问字符串中的单个字符的操作，要求索引类型为整数类型（`byte`/`short`/`int`）。字符串索引返回 `string` 类型（单字符子串）。本测试验证索引类型合法性、字符串不可变性、运行时行为以及跨语言（ArkTS/Java/Swift）行为差异。

## 三级测试点设计

### compile-pass（验证编译通过 + 语义正确）

| # | 测试点 | 说明 |
|---|--------|------|
| 001 | int 字面量索引 | `s[0]`, `s[1]`, `s[2]` — int 字面量作为索引 |
| 002 | int 变量索引 | `s[idx]` — int 变量作为索引 |
| 003 | byte 索引 | `s[b]` — byte 隐式加宽到 int 作为索引 |
| 004 | short 索引 | `s[sh]` — short 隐式加宽到 int 作为索引 |
| 005 | 首尾索引 | `s[0]`, `s[len-1]` — 字符串首尾字符访问 |
| 006 | long→.toInt() 转换 | long 类型通过 `.toInt()` 显式转换后索引 |
| 007 | 链式调用索引 | `getStr()[0]` — 方法返回值链式索引 |

### compile-fail

| # | 测试点 | 说明 |
|---|--------|------|
| 008 | 字符串不可变性修改 | `s[1] = "d"` — 编译时错误，与数组区分 |
| 009 | string 作为索引 | `s["abc"]` — string 类型不可作为索引 |
| 010 | float 字面量索引 | `s[3.5]` — 浮点类型不可作为索引 |
| 011 | long 无转换索引 | `s[l]` — long 无显式转换不可作为索引 |
| 012 | float 无转换索引 | `s[f]` — float 无转换不可作为索引 |
| 013 | double 无转换索引 | `s[d]` — double 无转换不可作为索引 |

### runtime（验证字符串索引运行时行为）

| # | 测试点 | 预期值 |
|---|--------|--------|
| 014 | 字符访问正确性 | 首字符 `s[0]` → "a"，中间字符 `s[2]` → "k"，尾字符 `s[4]` → "s" |
| 015 | 变量索引运行时 | int 变量索引访问字符正确性 |
| 016 | 越界（索引 >= 长度） | `s[10]` (len=3) → RangeError |
| 017 | 负索引 | `s[-1]` (len=3) → RangeError |

## 文件命名规范

- YYY: 001 起连续编号
