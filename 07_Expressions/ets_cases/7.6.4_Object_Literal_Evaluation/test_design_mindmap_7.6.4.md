# 7.6.4 Object Literal Evaluation - 测试设计思维导图

## 概述

对象字面量求值定义了运行时语义：构造器先执行 → 字段从左到右求值与赋值 → 异常传播。

## 三级测试点设计

### compile-pass（验证编译通过 + 语义正确）

| # | 测试点 | 说明 |
|---|--------|------|
| 001 | 命名类对象字面量 | 命名类类型对象字面量编译通过 |
| 002 | 接口类型字面量 | 接口类型对象字面量编译通过 |
| 003 | 多字段字面量 | 多字段对象字面量编译通过 |
| 004 | 字段含表达式 | 字段含表达式（函数调用、算术运算）编译通过 |
| 005 | 构造器+字段覆盖 | 构造器执行+字段覆盖编译通过 |

### compile-fail

| # | 测试点 | 说明 |
|---|--------|------|
| （无） | — | 本节无 compile-fail 用例 |

### runtime（验证实际运行时行为）

| # | 测试点 | 预期值 |
|---|--------|--------|
| 006 | 正常完成：构造器执行+字段赋值正确 | 构造器执行后字段值正确 |
| 007 | 字段从左到右求值顺序验证 | 全局追踪器验证 "A→B→C" 顺序 |
| 008 | 构造器抛异常求值终止 | @runtime-throws=Error |
| 009 | 字段表达式抛异常求值终止 | @runtime-throws=Error |
| 010 | 构造器先执行再字段赋值（覆盖构造器值） | 字段值覆盖构造器默认值 |
| 011 | 接口匿名类构造器调用验证 | 构造器调用正确 |

## 文件命名规范

`EXP_07_06_04_YYY_{CATEGORY}_{DESCRIPTION}.ets`

- YYY: 001 起连续编号
- CATEGORY: PASS / FAIL / RUNTIME
- DESCRIPTION: 大写+下划线描述

## 注释模板要求

每个文件必须包含 5 个 JSDoc tag：@id, @expect, @section, @design, @note
