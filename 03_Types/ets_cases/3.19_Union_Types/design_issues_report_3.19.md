# 3.19 Union Types - 设计问题报告

## 一、本章节首次发现的设计问题

### 问题 1: 联合类型中函数类型不加括号不报错 — 与 3.18 章节相同问题

**问题描述**：
Spec 3.19 明确规定 "A compile-time error occurs if type is a function type not enclosed in parentheses"，与 3.18 章节相同的 SPEC 不一致问题。

**复现用例 ID**：TYP_03_19_006_FAIL_UNION_FUNCTION_NO_PARENS

**实测行为**：编译通过（与 Spec 矛盾）

**严重性等级**：MEDIUM（与 D-3.18-01 相同问题）

**分析**：此问题已在 3.18 章节中记录（D-3.18-01），此处仅确认 3.19 章节中的联合类型场景同样受影响。
